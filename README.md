# AI Resume Builder and Career Enhancement System

## Features
- Resume Builder
- Resume Preview
- Skill Gap Analysis
- JD Keyword Matching
- Intelligent Project Recommendations
- Interview Question Suggestions
- Resume PDF Download
- Resume DOC Download
- OTP Password Reset
- Login & Signup System

## Tech Stack
- Python
- Django
- SQLite
- HTML
- CSS
- JavaScript

## Description
AI-powered Resume Builder that analyzes resumes and job descriptions, identifies missing skills, and recommends projects and interview preparation resources based on job roles.
